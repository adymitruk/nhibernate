===============================================================================
README for NHibernate.Contrib
===============================================================================

NHibernate.Contrib are a number of projects that complements the NHibernate 
core functionalities.


What is "Contrib"
=================

The NHibernate.Contrib project is created to host extra functionalities
and additional pluggable components (e.g. extra dialect, proxy generators,
or HQL parsers).


How to build NHibernate.Contrib
===============================

1.  Build the NHibernate core first: In the "nhibernate" folder, run 
    "nant build".
2.  Run "nant build" in the "contrib" folder.
